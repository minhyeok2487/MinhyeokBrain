# 자료구조(Data Structure)
- 자료(Data)의 집합
- 각 원소들이 논리적으로 정의된 규칙에 의해 나열되며 
  자료에 대한 처리를 효율적으로 수행할 수 있도록 자료를 구분하여 표현 하는것
- 메모리 공간을 효율적으로 사용해야 하는데 필요한 것
- 실행 시간의 효율성도 따짐
